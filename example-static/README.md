### example-static

This sample "app" is just an `index.html` file with "Hello World" text. It's staged using the built-in `static-site` stager.

To create the app, perform the following:

```
cd example-static
apc app create example-static-app
```

Next, start the app as follows:

```
apc app start example-static-app
```

Navigate to the URL provided from the app staging process to view the output page.
